# 🛍 E-Commerce Backend API

A RESTful backend service to manage Products, Categories, and Cart operations.

## 🚀 Hosted API
Base URL: `https://ecommerce-backend.onrender.com`

## 📦 Tech Stack
- Node.js + Express
- Deployed on Render

## 📚 API Endpoints

### Products
- `GET /products`
- `GET /products/:id`
- `POST /products`
- `PUT /products/:id`
- `DELETE /products/:id`

### Categories
- `GET /categories`
- `POST /categories`

### Cart
- `POST /cart`
- `GET /cart`
- `DELETE /cart`
